Administrar Pruebas

Esta ventana llamada "Administrar Pruebas" contiene un panel de búsqueda por número o nombre de prueba, un panel para registrar una nuevaprueba y un panel de botones.

Al presionar el botón "Nuevo" se oculta el panel de búsqueda y se habilita el campo nombre para ingresar una nueva prueba, luego se presiona el botón "Guardar" para registrar el dato ingresado.

El botón "Cancelar" borra lo ingresado en el campo nombre y aparece el panel búsqueda.

Para realizar una búsqueda de una prueba se debe seleccionar una opción de búsqueda, estas son por número o por nombre de prueba y presionar el botón "Buscar", si la búsqueda arrojo resultados se completara la tabla con los mismos.

Para editar o eliminar una ocupación o profesión se debe buscar y seleccionar con doble clic la que se desea modificar o eliminar. El botón "editar" habilita el campo nombre para modificar su contenido y el botón "eliminar" borra el registro seleccionado.

Al presionar el botón "Informe" aparece la siguiente ventana que nos permite obtener un listado con todas las pruebas registradas o filtrar el listado según una descripción.

Los datos requeridos para generar el listado son Ordenar por un campo que se mostrara en el listado, el tipo de ordenamiento del campo elegido y la cantidad de registros ha mostrar.
