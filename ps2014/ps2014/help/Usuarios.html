Administrar Usuarios

Esta ventana llamada "Administrar Usuarios" contiene un panel de búsqueda por número o nombre de usuario, un panel para registrar nuevo usuario y un panel de botones.

Al presionar el botón "Nuevo" se oculta el panel de búsqueda y se habilita los campos usuario y clave para ingresar un nuevo usuario, luego se presiona el botón "Guardar" para registrar el dato ingresado.

El botón "Cancelar" borra lo ingresado en el campo nombre y aparece el panel búsqueda.

Para realizar una búsqueda de un usuario se debe seleccionar una opción de búsqueda, estas son por número o por nombre de usuario y presionar el botón "Buscar", si la búsqueda arrojo resultados se completara la tabla con los mismos.

Para editar o eliminar un usuario se debe buscar y seleccionar con doble clic el que se desea modificar o eliminar. El botón "editar" habilita el campo nombre para modificar su contenido y el botón "eliminar" borra el registro seleccionado.
